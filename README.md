# Central Técnica Leite

Arquivos públicos do formulário da Central Técnica Leite.

## Como atualizar no GitHub

1. Extraia este ZIP.
2. No repositório `central-tecnica-leite`, clique em `Add file > Upload files`.
3. Suba todos os arquivos e a pasta `assets`, substituindo os existentes.
4. Faça commit.
5. Aguarde 1 a 3 minutos e atualize a página publicada.

O arquivo `config.js` já está com o endpoint do Google Apps Script configurado.
